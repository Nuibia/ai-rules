# 仓库概览

本仓库旨在收集可跨项目复用的 AI 协作规范与工具，包括提交信息规则、AI 助手指令模板、可复制的配置文件以及配套文档站点。你可以通过以下方式快速上手：

- 浏览 `guides/`、`docs/` 下的 Markdown 文档，了解规范细节。  
- 运行 `yarn docs:dev` 启动 dumi 文档站点，获得更友好的阅读体验。  
- 使用后续提供的 CLI 命令，将规范文件自动写入目标项目中。  

后续的 CLI 开发进展会同步在「CLI 使用」章节，敬请期待。若你希望贡献新的规范或模板，欢迎提交合规的 Pull Request。记得遵循提交信息规范并附带必要的测试说明。
